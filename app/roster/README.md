# Gacha Roster Bot

Сезонная коллекционная игра в Telegram.

Игроки собирают набор карточек персонажей, разделённых на команды, фракции или тематические группы. Каждая карточка имеет собственную редкость и может выпасть случайным образом при броске.

## Основные механики

### 🎲 Роллы

Команда `/roll` выдаёт случайную карточку из активного сезона.

На броски действуют:

* кулдаун между попытками;
* лимит роллов за два часа;
* дневной лимит роллов.

Вероятность выпадения зависит от редкости карточки.

### 📚 Коллекция

Каждый игрок собирает собственный альбом.

Карточки объединены в команды. Для каждой команды отображается прогресс сбора и список уже открытых персонажей.

Неоткрытые карты скрываются до получения.

### 🔄 Дубликаты

Повторные выпадения не пропадают.

Все дубликаты сохраняются и используются как ресурс для обмена.

### ✨ Крафт

Накопив достаточное количество дубликатов, игрок может использовать команду `/roll_craft`.

Крафт расходует дубликаты и гарантированно выдаёт новую карточку, которой ещё нет в коллекции игрока.

Если коллекция полностью собрана, крафт становится недоступен.

### 🏆 Завершение коллекции

Цель сезона — собрать максимальное количество уникальных карточек до окончания события.

Игрок может:

* закрывать отдельные команды на 100%;
* собирать полный альбом сезона;
* оптимизировать использование дублей через крафт.

### 📊 Профиль игрока

Команда `/me` показывает:

* прогресс по командам;
* количество собранных карточек;
* доступные броски;
* текущий кулдаун;
* оставшееся время сезона.

## Игровой цикл

Ролл → Новая карта или дубликат → Пополнение коллекции → Накопление дублей → Крафт новой карты → Закрытие коллекций → Завершение сезона.
