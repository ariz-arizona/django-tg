# 🔮 Tarot Bot

Телеграм-бот для раскладов Таро и Оракулов.

## 🚀 Быстрый старт

```
/start    — Начать работу с ботом
/help     — Список всех команд
```

---

## 🃏 Основные команды

### `/card` — Случайная карта
Одна случайная карта из стандартной колоды.

```
/card
```

### `/cardN` — Расклад на N карт
N случайных карт. Максимум — 10.

```
/card3      # 3 карты
/card9      # 9 карт
/card10     # 10 карт
/card100    # Всё равно 10 (ограничение)
```

### `/card_flip` — С перевернутыми картами
То же самое, но карты могут быть перевёрнутыми.

```
/card_flip       # 1 перевернутая
/card flip       # то же самое (пробел)
/card9_flip      # 9 перевернутых
/card9 flip      # то же самое
```

---

## 🎴 Старшие арканы

### `/card_major` — Только старшие арканы
Только 22 карты старших арканов (без младших).

```
/card_major          # 1 старший аркан
/card major          # то же самое
/card9_major         # 9 старших арканов
/card9_major_flip    # 9 старших арканов + перевернутые
/card9 major flip    # то же самое (пробелы)
```

---

## 🎯 Конкретные позиции

### `/cardN cX_Y_Z` — Карты по номерам
Запрос конкретных карт по их позициям в колоде (с 0).

```
/card3_c0_1_2     # Первые 3 карты: Шут, Маг, Жрица
/card3 c0_1_2     # то же самое (пробел)
/card3_c3_4_5     # Карты 3, 4, 5
```

**Особенность:** одинаковые запросы всегда возвращают одинаковые карты.

## 📚 Колоды

### `/decks` — Список всех колод
Показывает доступные колоды с пагинацией.

```
/decks
```

### `/decks oraculum` — Список оракулов
Только колоды-оракулы:

```
/decks oraculum
```

---

## 🎨 Колоды по ID и слагу

### `/card_deck_N` — Расклад из колоды по ID
Где N — числовой ID колоды из `/decks`.

```
/card_deck_29        # 1 карта из колоды №29
/card9_deck_29       # 9 карт из колоды №29
/card9_deck_29_flip  # 9 перевернутых из колоды №29
```

### `/card_deck_SLUG` — Расклад из колоды по слагу
SLUG — текстовый идентификатор колоды.

```
/card_deck_illuminati      # 1 карта из Таро Иллюминатов
/card9_deck_illuminati     # 9 карт из Таро Иллюминатов
/card_deck_lenormand       # 1 карта Ленорман
```

### `/oraculum_deck_SLUG` — Оракул по слагу
Альтернативная команда для оракулов (результат тот же, что и `/card_deck_SLUG`).

```
/oraculum_deck_abandoned    # Заброшенный Оракул
/oraculum_deck_lenormand    # Ленорман
/oraculum_deck_angelarium   # Ангеларий
```

### 🆔 ID и слаг — одно и то же

**Команды по ID и по слагу взаимозаменяемы и дают одинаковый результат:**

```
/card_deck_29                # по ID
/card_deck_illuminati        # по слагу
→ Оба выдают карты из колоды «Таро Иллюминатов»

/card9_deck_abandoned         # по слагу
/card9_deck_42                # по ID (если 42 — Заброшенный Оракул)
→ Оба выдают 9 карт из одной колоды
```

**Форматы одной и той же команды:**
```
/cardN_deck_ID
/cardN_deck_SLUG
/cardN_deck_ID_flip
/cardN_deck_SLUG_flip
/cardN_deck_ID_major
/cardN_deck_SLUG_major
/cardN_deck_ID_major_flip
/cardN_deck_SLUG_major_flip
```

## 🔄 Режим «Вся колода»

При нескольких нажатиях «Еще карту» бот предлагает переключиться в режим просмотра всей колоды:

```
💡 Большие расклады удобнее смотреть в режиме «Вся колода»: 
/all_deck_29
```

**Форматы `/all_deck`:**
```
/all_deck_N              — вся колода по ID
/all_deck_N_flip         — вся колода + перевернутые
/all_deck_N_major        — только старшие арканы
/all_deck_N_major_flip   — старшие арканы + перевернутые
```

## 🎮 Кнопки в сообщениях

| Кнопка | Действие |
|---|---|
| `Еще карту` | Новая случайная карта из той же колоды |
| `Трактовка (N)` | Получить трактовку для текущего расклада |
| `⬅️ Назад` / `➡️ Вперед` | Пагинация в списке колод |

---

## 📊 Прогресс колоды

При раскладах показывает прогресс:

```
Всего в колоде: 11/78
```

- **Стандартная колода:** 78 карт
- **Старшие арканы:** 22 карты
- **Оракулы:** разное количество

## 📝 Форматы команд

Большинство команд поддерживают запись и через `_`, и через пробел:

```
/card_flip              ✅
/card flip              ✅
/card9_major            ✅
/card9 major            ✅
/card9_deck_29_flip     ✅
/card9 deck 29 flip     ✅
```

## 🔧 Техническое

- Бот запоминает выбранную колоду для этого гадания
- Карты не повторяются, пока колода не кончится
- ID и слаг колоды взаимозаменяемы — можно использовать что удобнее
